## build from source

```bash
git clone https://github.com/senderpa/4ma-app.git && cd 4ma-app
npm install
npm run tauri dev     # dev mode
npm run tauri build   # create installer
```

needs: [Rust](https://rustup.rs), [Node 18+](https://nodejs.org), [Tauri prerequisites](https://tauri.app/start/prerequisites/)
